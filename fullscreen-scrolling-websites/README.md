# Fullscreen Scrolling Websites

## feature
- Infinite Scroll websites
- 一頁式向下卷動頁面
- 可以使用 ↓, ↑, page up, page down, home, end 切換頁面
- 可以使用 滑鼠卷動頁面

## dependency
- bootstrap
- jQuery

## reference
- [See document](https://github.com/alvarotrigo/fullPage.js)

## view
```bash
php -S localhost:9000
```
